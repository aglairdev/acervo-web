## Config

Para o funcionamento do motor de busca, é necessário popular o diretório `data/` com arquivos JSON seguindo a estrutura abaixo:

```json
[
  {
    "name": "",
    "console": "",
    "size": "",
    "link": ""
  }
]
```
Também precisa do arquivo `index.json`:

```json
{
  "consoles": ["", ""]
}
```
